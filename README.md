## Deskripsi Project
Project ini merupakan aplikasi To-Do List sederhana berbasis web.  
Aplikasi ini dibuat sebagai **tugas tambahan**, bukan UAS, untuk melatih penggunaan HTML, CSS, dan JavaScript (DOM).

## Alur Logika / Algoritma
1. Mengisi nama tugas
2. Sistem memvalidasi input (tidak boleh kosong)
3. Jika valid, tugas ditambahkan ke daftar
4. Pengguna dapat menandai tugas selesai atau menghapus tugas
5. Jumlah tugas total dan selesai diperbarui secara otomatis
